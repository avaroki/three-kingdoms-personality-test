# 삼국지 성격 유형 검사

삼국지 인물 중 나와 가장 닮은 캐릭터를 찾아보는 간단한 성격 유형 테스트입니다.

## 사이트

https://avaroki.github.io/three-kingdoms-personality-test/

## 소개

총 7개의 질문에 답하면 선택한 답변에 따라 삼국지 인물별 점수가 계산됩니다.  
가장 높은 점수를 받은 인물이 결과로 나오며, 캐릭터 이미지와 간단한 설명을 함께 확인할 수 있습니다.

## 등장 인물

- 유비
- 조조
- 제갈량
- 관우
- 장비
- 손권
- 여포
- 조운
- 주유
- 사마의
